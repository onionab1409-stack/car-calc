## Исследование · 27 февраля 2026

---

## 📌 Ключевые правила (с 01.12.2025)

**Постановление Правительства РФ от 01.11.2025 № 1713** — новая методика расчёта.

### Формула
```
Утильсбор = Базовая ставка × Коэффициент
```
- Базовая ставка для легковых (М1) = **20 000 ₽**
- Коэффициент зависит от: объём двигателя + мощность + возраст

### Льготный vs Коммерческий

| Тип | Условие | Сумма |
|-----|---------|-------|
| **Льготный (физлицо, личное пользование)** | ДВС ≤160 л.с. И объём ≤3.0L | 3 400₽ (до 3 лет) / 5 200₽ (3+ лет) |
| **Льготный электро** | 30-мин мощность ≤80 л.с. | 3 400₽ / 5 200₽ |
| **Коммерческий** | ДВС >160 л.с. ИЛИ объём >3.0L ИЛИ юрлицо ИЛИ перепродажа <1 год | От 307 200₽ до 4 572 000₽ |

### Когда физлицо платит КОММЕРЧЕСКИЙ утильсбор:
1. Мощность двигателя >160 л.с. (117,68 кВт)
2. Объём двигателя >3.0L (даже если ≤160 л.с.)
3. Продажа авто в течение 12 месяцев после ввоза
4. Второе авто за год (первое — по льготной, второе — коммерческая)
5. Оформление по ДТ (как юрлицо) вместо личного пользования
6. Электро/послед. гибрид: 30-мин мощность >80 л.с.

## 🔧 ПРИМЕНЕНИЕ В КАЛЬКУЛЯТОРЕ

### Алгоритм определения утильсбора:

```typescript
function getUtilSbor(car: CarInput): number {
  const { horsePower, engineVolumeCc, engineType, ageCategory } = car;
  
  // 1. Электромобили — отдельная таблица
  if (engineType === 'electric' || engineType === 'sequential_hybrid') {
    const power30min = car.power30min ?? horsePower * 0.45;
    if (power30min <= 80) return ageCategory === 'new' ? 3400 : 5200;
    return lookupElectricTable(power30min, ageCategory);
  }
  
  // 2. Обычные гибриды — суммарная мощность всех двигателей
  // Если суммарная ≤160 и объём ≤3000 → льготный
  
  // 3. ДВС: проверяем льготу
  if (horsePower <= 160 && engineVolumeCc <= 3000) {
    return ageCategory === 'new' ? 3400 : 5200;
  }
  
  // 4. Коммерческий — ищем в таблице по объёму + мощности + возрасту
  return lookupCommercialTable(engineVolumeCc, horsePower, ageCategory);
}
```

### Диапазоны объёма двигателя:
1. `≤1000` — до 1.0L
2. `1001–2000` — 1.0–2.0L (самый частый)
3. `2001–3000` — 2.0–3.0L
4. `3001–3500` — 3.0–3.5L (⚠️ коммерческий ВСЕГДА, даже ≤160лс)
5. `>3500` — >3.5L (аналогично, ещё дороже)

### Диапазоны мощности (шаг ~30 л.с.):
```
≤160 | 160–190 | 190–220 | 220–250 | 250–280 | 280–310 | 
310–340 | 340–370 | 370–400 | 400–430 | 430–460 | 460–500 | >500
```

### Возраст:
- `new` = до 3 лет с даты выпуска
- `used` = 3+ лет

## 📋 Входные данные для расчёта

Для утильсбора нужно знать:
1. **Мощность** (л.с.) — обязательно
2. **Объём двигателя** (см³) — обязательно  
3. **Тип двигателя** — ДВС / электро / гибрид / послед. гибрид
4. **Возраст** — до 3 лет / 3+ лет
5. **Кто ввозит** — физлицо (личное) / юрлицо (коммерческое)

> ⚠️ В нашем калькуляторе для авто ≤160 л.с. утильсбор (3400/5200₽) уже ВКЛЮЧЁН в ФИКС.
> Доплата утильсбора — ТОЛЬКО для >160 л.с. (или >3.0L, или электро >80лс).
> Эта доплата = коммерческий утильсбор из таблицы МИНУС льготный (3400/5200₽).

### Формула доплаты:
```
utilSborExtra = commercialUtilSbor - (ageNew ? 3400 : 5200)
```
