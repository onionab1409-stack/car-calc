## 1. Когда нужен ТКС / ЕТТ ЕАЭС

В нашем проекте для авто **до 3 лет** — используются фиксированные коэффициенты (× 1.48 для РФ, × 1.30 для РБ).

**ТКС нужен для:**
1. Авто **3–5 лет** — вместо коэффициентов нужен точный расчёт по ЕТТ
2. Расчёт **утильсбора** при мощности >160 л.с.

## 3. Формула расчёта для авто 3–5 лет (физлица)

```
Пошлина = объём_двигателя_см³ × ставка_за_см³ × курс_EUR/RUB

Пример: Toyota Camry 2021, 2500 см³, 3-5 лет
Ставка: 3.0 €/см³ (диапазон 2301–3000)
Пошлина: 2500 × 3.0 = 7,500 € × ~100₽ = 750,000₽
```

**Это ВМЕСТО коэффициента × 1.48 / × 1.30.**

В наших формулах для 3–5 лет:
```
USA → РФ (3-5 лет):
  Базовая = (Лот × 1.08 + $2200 + $750) × 1.011
  Пошлина_ЕТТ = engineCC × rate_per_cc × EUR_RUB
  Итого = (Базовая + $2200) × USDT_rate + Пошлина_ЕТТ + ФИКС

Korea → РФ (3-5 лет):
  Итого = Цена_KRW × KRW_rate + 90,000₽ + Пошлина_ЕТТ + 600,000₽

China → РФ (3-5 лет):
  Итого = (Цена + 8000¥ + 2.5%) × CNY_rate + Пошлина_ЕТТ + 590,000₽
```

## 4. Утилизационный сбор

### Формула
```
Утильсбор = Базовая_ставка × Коэффициент
```

### Базовая ставка
- Легковые (некоммерческие, для себя): **20,000 ₽**
- Коммерческие (юрлица, перепродажа): **150,000 ₽**

### Коэффициенты для физлиц (для себя, ≤160 л.с.)

| Возраст | Коэффициент | Утильсбор |
|---------|-------------|-----------|
| до 3 лет | 0.17 | 3,400₽ |
| старше 3 лет | 0.26 | 5,200₽ |

### ⚠️ С 1 декабря 2025: авто >160 л.с. = КОММЕРЧЕСКИЙ утильсбор

С 1.12.2025 льгота отменена. Для авто >160 л.с. независимо от физ/юрлица:

| Объём двигателя | До 3 лет (К) | Утильсбор | Старше 3 лет (К) | Утильсбор |
|-----------------|--------------|-----------|-------------------|-----------|
| до 1000 см³ | 4.06 | 609,000₽ | 10.36 | 1,554,000₽ |
| 1001-2000 | 15.03 | 2,254,500₽ | 26.44 | 3,966,000₽ |
| 2001-3000 | 42.24 | 6,336,000₽ | 63.95 | 9,592,500₽ |
| 3001-3500 | 48.50 | 7,275,000₽ | 74.25 | 11,137,500₽ |
| >3500 | 61.76 | 9,264,000₽ | 81.19 | 12,178,500₽ |

⚠️ **ВНИМАНИЕ:** Эти ставки ОГРОМНЫЕ. Для авто с 2.5л двигателем >160лс — утильсбор ~6.3М₽!

### Реальная ситуация для нашего проекта

Большинство наших клиентов:
- **≤160 л.с.** → утильсбор = 3,400₽ или 5,200₽ (копейки, включены в ФИКС)
- **>160 л.с.** → нужен точный расчёт по таблице коммерческих коэффициентов

Нам нужна таблица мощности → объёма для определения правильного коэффициента. Но в формулах пользователь указывает и объём, и мощность, так что можно определить точно.

## 5. Варианты реализации ТКС

### Вариант A: Парсинг tks.ru (❌ НЕ РЕКОМЕНДУЕТСЯ)
- tks.ru — JS-калькулятор, рендерится на клиенте
- Нужен headless browser (Puppeteer) → тяжело, медленно, ломается
- Нет публичного API
- Коммерческое API (calc.tks.ru) — платное, непонятные условия

### Вариант B: Python-библиотека tks-api-official (⚠️ МОЖНО, НО...)
- GitHub: markolofsen/tks-api (MIT лицензия)
- Все формулы уже реализованы на Python
- НО: нужен Python runtime, не TypeScript
- Можно использовать как справочник для своей реализации

### Вариант C: Свои формулы на TypeScript (✅ РЕКОМЕНДУЕТСЯ)
- Все ставки из ЕТТ ЕАЭС — публичные, в этом документе
- Таблицы фиксированные, обновляются раз в год
- Полный контроль, без внешних зависимостей
- Обновляем JSON через админку бота если ставки изменились

### Вариант D: calcus.ru API (⚠️ АЛЬТЕРНАТИВА)
- Современный калькулятор с актуальными ставками 2026
- Нужно проверить наличие API
- Fallback если свои формулы дали неверный результат

**РЕШЕНИЕ: Вариант C** — свои формулы. Используем tks-api как справочник.

## 6. TypeScript реализация (план для P3)

```typescript
// Ставки ЕТТ ЕАЭС для авто 3-5 лет (физлица)
const ETC_RATES_3_5: { maxCC: number; rateEUR: number }[] = [
  { maxCC: 1000,  rateEUR: 1.5 },
  { maxCC: 1500,  rateEUR: 1.7 },
  { maxCC: 1800,  rateEUR: 2.5 },
  { maxCC: 2300,  rateEUR: 2.7 },
  { maxCC: 3000,  rateEUR: 3.0 },
  { maxCC: Infinity, rateEUR: 3.6 },
];

// Ставки ЕТТ ЕАЭС для авто 5+ лет
const ETC_RATES_5_PLUS: { maxCC: number; rateEUR: number }[] = [
  { maxCC: 1000,  rateEUR: 3.0 },
  { maxCC: 1500,  rateEUR: 3.2 },
  { maxCC: 1800,  rateEUR: 3.5 },
  { maxCC: 2300,  rateEUR: 4.8 },
  { maxCC: 3000,  rateEUR: 5.0 },
  { maxCC: Infinity, rateEUR: 5.7 },
];

function calculateETCDuty(engineCC: number, ageCategory: '3to5' | 'over5', eurRubRate: number): number {
  const rates = ageCategory === '3to5' ? ETC_RATES_3_5 : ETC_RATES_5_PLUS;
  const bracket = rates.find(r => engineCC <= r.maxCC)!;
  return engineCC * bracket.rateEUR * eurRubRate;
}
```

## 7. Курс EUR/RUB

Для расчёта пошлин по ЕТТ нужен курс евро.

```
Источник: ЦБ РФ (уже есть в P2.1)
data.Valute.EUR.Value = 91.2965₽ (на 28.02.2026)
```

Добавить в модуль rates:
```typescript
EUR_RUB: data.Valute.EUR.Value / data.Valute.EUR.Nominal
```
