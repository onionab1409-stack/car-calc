## 🇺🇸 ФОРМУЛЫ: США

### USA → РФ (до 3 лет, ≤160 л.с.) ✅

```
Итого_USD = (Лот × 1.08 + $2200 + $750) × 1.011 + (Лот × 1.08 + $2200) × 0.48
Итого_RUB = Итого_USD × USDT_rate + ФИКС
```

**Разбивка:**
| Компонент | Формула | Описание |
|-----------|---------|----------|
| Аукционный сбор | Лот × 0.08 | 8% от цены лота |
| Морская доставка | $2,200 | фиксированная |
| Страховка | +$750, затем × 1.011 | 1.1% от всей суммы |
| Растаможка РФ | (Лот × 1.08 + $2200) × 0.48 | 48% от (авто + доставка) |
| Фикс | см. таблицу ниже | СБКТС, ЭПТС, брокер и т.д. |

**Фиксированные суммы USA → РФ:**

| Цена лота (USD) | Фикс (₽) |
|------------------|-----------|
| ≤ $20,000 | 425,000 |
| $20,001 – $30,000 | 495,000 |
| $30,001 – $40,000 | 575,000 |
| > $40,000 | 575,000 + 100,000₽ за каждые $10,000 сверх $40K |

### USA → РБ (до 3 лет, ≤160 л.с.) ✅

```
Итого_USD = (Лот × 1.08 + $2200 + $750) × 1.011 + (Лот × 1.08) × 0.30
Итого_RUB = Итого_USD × USDT_rate + ФИКС
```

**Отличие от РФ:** растаможка 30% (не 48%) и считается от (Лот × 1.08) без $2200.

**Фиксированные суммы USA → РБ:**

| Цена лота (USD) | Фикс (₽) |

| $30,001 – $40,000 | 600,000 |
| > $40,000 | 600,000 + 100,000₽ за каждые $10,000 сверх $40K |

### USA → РФ (3–5 лет) ✅

```
Базовая формула (как USA → РФ до 3 лет) + ТКС вместо 48%
```

## ⚡ ДОПЛАТА ЗА МОЩНОСТЬ >160 Л.С.

### Правило
Если мощность авто > 160 л.с. → к итоговой цене добавляется утилизационный сбор.

### Источник
- Постановление РФ №1713
- С 1 декабря 2025 года льгота отменена для авто >160 л.с.
- Парсинг tks.ru / calcus.ru для актуальных ставок

### Ориентировочные ставки (коммерческий утильсбор)

| Мощность (л.с.) | Утильсбор (₽) |
|------------------|----------------|
| 161 – 190 | ~750,000 |
| 191 – 220 | ~794,000 |
| 221 – 250 | ~1,200,000 |
| 251 – 300 | ~1,500,000 |
| > 300 | ~2,000,000+ |

⚠️ **Точные ставки будут уточнены в P2 (Исследователь, подэтап 2.3).**

### Логика в коде

```typescript
function getUtilSbor(horsePower: number): number {
  if (horsePower <= 160) return 0;
  // Найти ставку из таблицы utilSbor (AdminSettings)
  // Если не найдено → парсинг tks.ru
  // Если парсинг упал → fallback из JSON
}
```

## 🔧 ТКС (Таможенный калькулятор)

### Что это
ТКС = таможенный калькулятор на tks.ru — полный расчёт растаможки по формулам ЕТТ ЕАЭС.

### Когда используется
1. Авто **3–5 лет** (вместо фиксированных коэффициентов 0.48 / 0.30)
2. Расчёт **утильсбора** для авто >160 л.с.

### Fallback
Если tks.ru недоступен → свои формулы ЕТТ ЕАЭС (реализуются в P3, подэтап 3.5).

### Кэширование
TTL = 24 часа (одни и те же параметры → один результат).
