## 🎨 P5 · ДИЗАЙНЕР — НОВЫЙ WORKFLOW

### Принятое решение: Midjourney + Spline + Claude

**Проблема:** Midjourney даёт красивые картинки, но при переводе картинка→код теряются детали и уходит много времени.

**Решение — 3D UI через Spline:**
1. **Midjourney** → генерация визуальных референсов (настроение, стиль, палитра, фоны)
2. **Пользователь** скидывает картинку в чат
3. **Claude** смотрит картинку → пишет точный промпт для Spline AI
4. **СТОП** → ждёт «продолжай»
5. **Пользователь** делает в Spline по промпту
6. **Экспорт** из Spline → React-компонент (`@splinetool/react-spline/next`)
7. **Claude** встраивает в Next.js проект

**Фоны** тоже генерируем в Midjourney → Claude пишет промпт → применяем.

### Интеграция Spline + Next.js:
```tsx
import Spline from '@splinetool/react-spline/next';

// Встраивание:
<Spline scene="https://prod.spline.design/xxxxx/scene.splinecode" />

// Или self-hosted .splinecode файл на VPS:
<Spline scene="/spline/country-selector.splinecode" />

// С обработкой событий:
<Spline
  scene="url"
  onSplineMouseDown={(e) => {
    if (e.target.name === 'USA') handleCountrySelect('US');
  }}
/>
```

## 📱 ВСЕ ЭКРАНЫ MINI APP (полная карта)

### Экран 1: ВЫБОР СТРАНЫ (главный, первое впечатление)
- 4 карточки: 🇺🇸 США | 🇰🇷 Корея | 🇦🇪 ОАЭ | 🇨🇳 Китай
- Hover/click анимации
- **→ Spline 3D** (главный wow-элемент)

### Экран 2: ВЫБОР НАПРАВЛЕНИЯ
- 2 кнопки: 🇷🇺 Россия | 🇧🇾 Беларусь
- **→ Spline 3D** (toggle-переключатели)

### Экран 3: ВВОД ДАННЫХ АВТО
- Поле: Цена (в валюте страны: $, ₩, AED, ¥)
- Поле: Год выпуска (выбор или ввод)
- Выбор: Тип двигателя (бензин / дизель / электро / гибрид)
- Поле: Мощность (л.с.)
- **→ Поля ввода = обычный CSS** (клавиатура Telegram, скорость)
- **→ Иконки двигателей = Spline 3D** (маленькие, акцентные)

### Экран 4: ЗАГРУЗКА (1-3 сек пока идёт расчёт)
- Анимация ожидания
- **→ Spline 3D** (low-poly машинка или абстрактная анимация)

### Экран 5: РЕЗУЛЬТАТ (ключевой экран!)
- **ОДНА БОЛЬШАЯ ЦИФРА** — итоговая цена в рублях
- Подпись "полная стоимость доставки"
- Кнопка "Оставить заявку"
- Кнопка "Новый расчёт"
- **→ Spline 3D** (премиальная 3D-карточка с ценой)

### Экран 6: ФОРМА ЗАЯВКИ
- Имя, Телефон, Комментарий
- Кнопка "Отправить"
- **→ Обычный CSS** (простая форма)

### Экран 7: ИСТОРИЯ РАСЧЁТОВ
- Список последних 20 расчётов (localStorage)
- Карточки: страна, авто, цена, дата
- **→ Обычный CSS** (список)

## ⚡ ОПТИМИЗАЦИЯ SPLINE ДЛЯ TELEGRAM

- **Простые формы** — rounded rect, pill, cube (минимум полигонов)
- **Мало частиц** — не больше 10 на сцену
- **Без текстур** — только material colors и gradients
- **Максимум 2 источника света** на сцену
- **.splinecode < 500KB** на элемент
- **Lazy loading** — Spline грузится только когда экран виден
- **Канвас** — 390×844 px (iPhone 14, стандарт Telegram Mini App)
- **Камера** — Orthographic (плоская, для UI)
- **Фон** — #0C0C0E или transparent
